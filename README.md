
# Mnelo

Mnelo is a premium SaaS interface for an AI workforce focused on MEP estimation and procurement.

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- App Router

## Routes

- `/` landing page
- `/login` login page
- `/dashboard` dashboard
- `/projects` projects page
- `/projects/[id]` project details and BOQ results

## Development

```bash
npm install
npm run dev
